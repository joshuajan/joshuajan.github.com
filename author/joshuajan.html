python的可变类型(mutable)和不可变类型(immutable)

Permalink: 2013-02-23 22:49:00+08:00 by JoshuaJan in python tags: python

Python的所有数据类型都是对象。 就算是变量也是对象。而对于python来说，对象主要有两大类型可变类型和不可变类型。这里需要使用is来判断两个对象是否指向同一个值。 不可变类型

可以做以下简单测试： a = 1 b = a a is b True 也就是a和b是指向同一个值，在python里面数字类型（int, float...）,sring, tuple都是不可变类型。但是如果再执行以下的代码会怎样呢？ a = a + 1 a is b False id(a) 165154980 >>> id(b) 165154992

当你改变这个int变量的数值的时候，其实python就是创建了一个新的int实例，通过 id() 可以查看他们的身份标识码，当你修改这个值的时候标识码就已经改变了。所以a is b就返回False。
